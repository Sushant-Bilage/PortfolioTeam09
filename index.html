About the DAA Subject

Design & Analysis of Algorithms (DAA) focuses on creating efficient solutions for real-world problems using well-defined algorithmic strategies. In this project, all Smart City ideas are designed strictly using algorithms and data structures taught in class.

Graph Algorithms – ~40% Real-World Importance

Algorithms such as Dijkstra, Bellman-Ford, Floyd-Warshall, Prim’s, Kruskal’s and Warshall’s are used to model cities as graphs. These are applied in road networks, water pipelines, waste collection routes, energy-efficient transport systems, and network optimization.

Trees (BST, AVL, Red-Black, 2-3 Trees) – ~25%

Tree structures allow fast searching, insertion, and deletion. They are used to store sports records, student performance, vehicle information, sensor data, and city infrastructure logs.

Sorting Algorithms – ~15%

Sorting algorithms such as Bubble, Selection, Insertion, Merge, Quick, and Heap Sort are used to rank players, prioritize services, analyze consumption data, and generate reports.

Heaps & Priority Queues – ~10%

Priority Queues manage scheduling problems such as emergency response, transport allocation, EV charging, and ranking systems in Smart City applications.

Hashing Techniques – ~10%

Hashing enables constant-time access to records. It is used for storing user data, sensor logs, authentication, and real-time system lookups.
