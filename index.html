Smart City Project – Design & Analysis of Algorithms

This project demonstrates the real-world application of Design and Analysis of Algorithms (DAA) in building intelligent Smart City solutions.

The ideas covered use core DAA concepts such as Graphs, Trees, Shortest Path Algorithms, Greedy Methods, Dynamic Programming, Sorting, Searching, and Hashing. These concepts are critical in solving optimization, scalability, and efficiency problems in real systems.

DAA Concepts Covered

✔ Graph Algorithms (BFS, DFS, Dijkstra, Prim, Kruskal, Floyd, Bellman-Ford)
✔ Trees (BST, AVL, Heap, Priority Queue)
✔ Sorting Algorithms (Merge, Quick, Heap)
✔ Hash Tables & Searching
✔ Greedy & Dynamic Programming Techniques

Smart City Applications

Each idea applies these algorithms to practical domains such as sports management, transport optimization, education accessibility, waste management, and water distribution.
